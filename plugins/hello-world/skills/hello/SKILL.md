# Hello

This skill exists to prove the `hello-world` plugin loaded from the
`seanodell` marketplace.

When invoked, confirm to the user that:

1. The `hello-world` plugin is installed and its skills are loading.
2. It came from the `seanodell` marketplace.
3. It is installed at **user scope**, so this skill is available in every project
   on the machine — not just this repo.

Then greet the user with a short, friendly hello.
